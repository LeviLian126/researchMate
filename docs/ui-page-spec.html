ResearchMate UI Page Spec

TLDR: UI 已有 Glacier 视觉方向。MVP 页面只包含 Ask、Library、Quiz 和 Admin Trace。普通用户不能看到 Developer Trace 入口。

2. 视觉方向

统一使用 Glacier, Glassmorphism：

  • 背景：#0a0e1a
  • 主色：#7dd3fc
  • 辅色：#c8a0f0
  • 面板：半透明玻璃、轻边框、低强度 glow
UI agent 可以调整具体布局，但不得把产品改成营销落地页。

4. Ask 页面

4.1 核心区域

  • 左侧：项目和会话列表
  • 右侧：消息流、回答、Sources panel
  • 底部：输入框、Slash command、Mode selector、上传快捷入口

4.2 Mode selector

只允许：

Auto / Local / Web / Hybrid

不允许：

  • Quiz
  • Report
  • Flashcards
  • Summary

4.3 Sources header

回答顶部只显示：

Mode: Hybrid
Sources: 5 local chunks · 3 web pages

详细来源放入 Sources panel。

5. Library 页面

5.1 文件状态

状态用户提示
uploaded文件已上传，等待解析
parsing正在解析
parsed已解析，等待索引
indexing正在索引
ready可用于 Ask
failed解析失败，可重试或删除
expired已过期
deleted已删除

5.2 错误态

  • 文件超 25MB：直接拒绝上传。
  • 类型不在 PDF/DOCX/PPTX：直接拒绝。
  • signed URL 过期：提示重新上传。
  • parser 失败：显示可读错误，不显示 stack trace。

6. Quiz 页面

6.1 显示内容

  • QuizSet 标题
  • 生成时间
  • Source summary
  • 选择题：题干、4 个选项、答案、解析
  • 简答题：题干、标准答案、解析

6.2 非目标

不做：

  • 在线答题
  • 自动批改
  • 错题本
  • 自适应题库

7. Admin Trace 页面

7.1 可见字段

  • trace_id
  • resolved_mode
  • task_type
  • router_reason
  • retrieved_chunks
  • tool_calls
  • validation_result
  • latency_ms
  • token_usage
  • 脱敏后的错误信息

7.2 禁止字段

  • API key
  • OAuth token
  • R2 signed secret
  • 完整 system prompt
  • 普通用户隐私 raw payload

8. 验收标准

  • [x] Mode selector 不包含 Quiz。
  • [x] Sources panel 能承载 local 和 web 两类来源。
  • [x] Library 状态覆盖上传、解析、索引、失败、删除。
  • [x] Quiz 有选择题和简答题渲染契约。
  • [x] Developer Trace 与普通用户导航隔离。

# ResearchMate UI Page Spec

> **TLDR:** UI 已有 Glacier 视觉方向。MVP 页面只包含 Ask、Library、Quiz 和 Admin Trace。普通用户不能看到 Developer Trace 入口。

## 1. 设计事实源

| 文件 | 说明 |
|---|---|
| `ui/chat/DESIGN.md` | Ask 页视觉方向 |
| `ui/library/DESIGN.md` | Library 页视觉方向 |
| `ui/quiz/DESIGN.md` | Quiz 页视觉方向 |
| `ui/*/code.html` | 主要界面参考 |
| `docs/ui-page-spec.html` | 页面行为事实源 |

## 2. 视觉方向

统一使用 **Glacier, Glassmorphism**：

- 背景：`#0a0e1a`
- 主色：`#7dd3fc`
- 辅色：`#c8a0f0`
- 面板：半透明玻璃、轻边框、低强度 glow

> UI agent 可以调整具体布局，但不得把产品改成营销落地页。

## 3. 页面清单

| 路由 | 页面 | 用户 | 说明 |
|---|---|---|---|
| `/` | 登录入口 | public | 只负责进入产品 |
| `/app` | Project list | 登录用户 | 创建和选择项目 |
| `/app/projects/:id` | Ask | owner | 统一问答入口 |
| `/app/projects/:id/library` | Library | owner | 上传、状态、删除 |
| `/app/projects/:id/quiz` | Quiz | owner | 查看 QuizSet |
| `/dev/traces/:trace_id` | Admin Trace | developer/admin | 不进入普通导航 |

## 4. Ask 页面

### 4.1 核心区域

- 左侧：项目和会话列表
- 右侧：消息流、回答、Sources panel
- 底部：输入框、Slash command、Mode selector、上传快捷入口

### 4.2 Mode selector

只允许：

```text
Auto / Local / Web / Hybrid
```

不允许：

- Quiz
- Report
- Flashcards
- Summary

### 4.3 Sources header

回答顶部只显示：

```text
Mode: Hybrid
Sources: 5 local chunks · 3 web pages
```

详细来源放入 Sources panel。

## 5. Library 页面

### 5.1 文件状态

| 状态 | 用户提示 |
|---|---|
| `uploaded` | 文件已上传，等待解析 |
| `parsing` | 正在解析 |
| `parsed` | 已解析，等待索引 |
| `indexing` | 正在索引 |
| `ready` | 可用于 Ask |
| `failed` | 解析失败，可重试或删除 |
| `expired` | 已过期 |
| `deleted` | 已删除 |

### 5.2 错误态

- 文件超 25MB：直接拒绝上传。
- 类型不在 PDF/DOCX/PPTX：直接拒绝。
- signed URL 过期：提示重新上传。
- parser 失败：显示可读错误，不显示 stack trace。

## 6. Quiz 页面

### 6.1 显示内容

- QuizSet 标题
- 生成时间
- Source summary
- 选择题：题干、4 个选项、答案、解析
- 简答题：题干、标准答案、解析

### 6.2 非目标

不做：

- 在线答题
- 自动批改
- 错题本
- 自适应题库

## 7. Admin Trace 页面

### 7.1 可见字段

- `trace_id`
- `resolved_mode`
- `task_type`
- `router_reason`
- `retrieved_chunks`
- `tool_calls`
- `validation_result`
- `latency_ms`
- `token_usage`
- 脱敏后的错误信息

### 7.2 禁止字段

- API key
- OAuth token
- R2 signed secret
- 完整 system prompt
- 普通用户隐私 raw payload

## 8. 验收标准

- [x] Mode selector 不包含 Quiz。
- [x] Sources panel 能承载 local 和 web 两类来源。
- [x] Library 状态覆盖上传、解析、索引、失败、删除。
- [x] Quiz 有选择题和简答题渲染契约。
- [x] Developer Trace 与普通用户导航隔离。
