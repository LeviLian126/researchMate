本地运行顺序

  1. 安装 Python 依赖：pip install -r requirements-dev.txt。
  2. 启动 API：python -m uvicorn researchmate_api.main:app --app-dir apps/api/src --reload --host 127.0.0.1 --port 8000。
  3. 安装前端依赖：npm install。
  4. 启动前端：npm --workspace @researchmate/web run dev。
  5. 打开 http://localhost:3000/app，使用 dev token 创建项目、上传资料、提问、生成 Quiz。

上线替换点

  • 将本地 dev token 替换为 Supabase Auth JWT 校验。
  • 将 in-memory store 替换为 Supabase Postgres、R2、Qdrant、Redis adapter。
  • 将 demo web evidence 替换为 SERPER/Tavily + JINA/Firecrawl provider。
  • 将 deterministic answer/quiz fallback 替换为配置好的 OpenAI-compatible LLM provider，并保留 schema validation。

停止条件

  • 需要 Auto 默认联网时必须重新做隐私和成本评审。
  • 需要修改 owner_user_id、resource ownership 或 trace visibility 时必须先更新安全测试。
  • 需要放入真实 API key、OAuth token 或 R2 secret 时必须停止，改用本地 env 或部署平台 secret manager。
