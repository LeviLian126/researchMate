ResearchMate OpenAPI Spec

本页面内嵌 infra/openapi/openapi.yaml 的当前内容，覆盖 Project、Document、Ask、Quiz、Sources、Developer Trace、Job 与 Health 接口。

Handoff API Guide Execution Plan Test Plan Security Checklist

关键验收点

  • 所有受保护路由显式声明 bearerAuth。
  • /api/v1/documents/upload-url、/api/v1/documents/{document_id}/complete 支持本地开发上传闭环。
  • /api/v1/ask、/api/v1/quiz、/api/v1/runs/{run_id}/sources 与 /api/v1/dev/traces/{trace_id} 完成可追溯学习闭环。
