  • 文档是实时状态，不保留历史堆叠。

Risks

风险与假设

  • 本地 deterministic answer/quiz 不代表真实 LLM 质量。
  • demo web evidence 不联网；真实 Search/Crawl 需要 SSRF、重定向、缓存和来源质量保护。
  • in-memory store 不适合生产；必须替换为 Supabase/R2/Qdrant/Redis。
  • Next 目前使用 canary 版本规避 PostCSS 漏洞；稳定版修复后应回到 stable。

Decision gates

停止条件

  • 需要提交真实 secret、token、signed URL 或生产日志时，停止并改用 Secret Manager。
  • 需要改变 Auto 联网默认行为时，停止并做隐私、成本和安全评审。
  • 需要改 owner isolation、RLS、signed URL 或删除清理时，先补隔离和生命周期测试。
  • 质量门或 handoff validation 失败时，不允许声称完成。
